# Icons

These are placeholder solid-color PNGs sized to Tauri's default bundler layout.
Replace with production logo renders before shipping. Generate the full set
(including `.icns` / `.ico`) with:

```bash
pnpm tauri icon path/to/source-1024.png
```

Required sizes for macOS `.icns`: 16, 32, 64, 128, 256, 512, 1024 (plus 2x variants).
